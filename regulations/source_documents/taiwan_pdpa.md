# Taiwan Personal Data Protection Act (PDPA)

## 個人資料保護法

### Article 6: Sensitive Personal Data

The following are considered sensitive personal data (敏感性個人資料):

1. **National ID Number** (身份證字號)
2. **Health records** (醫療紀錄)
3. **Medical history** (病歷)
4. **Genetic information** (基因資訊)

### Special Provisions

- Ages over 90 (超過90歲) require special protection
- Rare diseases (罕見疾病) with small populations need anonymization
- Hospital names combined with rare diagnoses can be identifying
